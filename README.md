# AlphaBet

## Установка

- Создать файл `.env` используя шаблон `.env-sample`;
- Значение для `SUPABASE_KEY` - `anon_key` из вывода команды: `npx supabase start --ignore-health-check`;
- Установить зависимости:

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Остановка:

```bash
npm run stop
```
